Wests Of Survival

Project Objective

Wests of Survival is a survival game created as part of a university project. I developed it with the help of Dulhoste Maxime and Mihoubi Maroune. We imagined the game's gameplay, maps, and various map designs. We used the C# programming language with the Monogame framework, and to design the map, we used Tiled.

Tiled Presentation

Monogame Presentation

MonoGame is an open-source game development framework that allows for creating cross-platform 2D and 3D games. MonoGame is a reimplementation of the Microsoft XNA framework.
