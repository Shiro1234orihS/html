    BMW Motorrad

    Le but du projet avec Laravel

    L'ojectif du projet BMW est de reproduire les différentes fonctionnalités du site web de BMW Motorrad.

    Création des documents

    Création de la base

    BMW Motorrad en Vue.js et une API en ASP.NET

    Le but du projet

    L'objectif du projet BMW est de reproduire le site web de BMW Motorrad, mais les différentes fonctionnalités du site web sont gérées par une API en ASP.NET avec le langage C#.

    Détails techniques pour l'API et création de la base

    L'API utilise Entity Framework pour la gestion des contrôleurs et pour la gestion des différents managers qui offrent les différentes fonctionnalités. La base de données est stockée sur un serveur MySQL et a été générée en code first.

    Détails techniques pour la partie client

    Le client a été créé avec le framework Vue.js et l'extension Pinia avec laquelle sont stockées les différentes informations envoyées par l'API au client.

    Informations sur le projet

    Le groupe a été formé en collaboration avec Tuong Vy Pham Huynh, Evan Lamy et Hugo Raguin. Lien du répertoire du projet : https://github.com/phtuongvy/SAE_BMW_API
