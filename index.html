Presentation du jeu

Reconstruction du site BMW Motorrad avec Laravel.

Presentation du projet

L'objectif du projet BMW est de reproduire les différentes fonctionnalités du site web de BMW Motorrad.

Les différentes logiciels et Frameworks utilisé

Pour les différentes documentations (cas d'utilisation, diagramme de séquence, etc.).
J'ai réalisé un MCD (Modèle Conceptuel de Données), un MLD (Modèle Logique de Données), ainsi qu'un script de base de données en utilisant PowerAMC.
La base de données a été construite avec la technologie PostgreSQL.
Pour la création du site, j'ai utilisé Laravel pour me connecter à la base de données, ce qui m'a permis d'afficher les différentes données aux différents utilisateurs.
